Projects

NIOS II Emulator

This is an emulator for the NIOS II processor. It is a web based application that is capable of simulating most of the functions of the NIOS II processor. The system has taken it a step further and added features of NIOS II development boards. These features are switches, a seven segment display, and inspector for memory addresses.

Clubnizer

Clubinizer is in the paper and pencil design stage. The current idea is a system that is used for organizing clubs and organizations, hint the name. I hope to make it featured packed that way it is capable of replacing many of the applications organizations may use.
